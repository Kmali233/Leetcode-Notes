

# 两数之和

https://leetcode.cn/problems/two-sum?envType=study-plan-v2&envId=top-100-liked



#### 方法一：暴力枚举

2个 for 循环

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        n = len(nums)
        for i in range(n):
            for j in range(i+1,n):
                if nums[i] + nums[j] == target:
                    return [i,j]
```





#### 方法二：哈希表

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        hashtable = dict()
        for i,num in enumerate(nums):
            if target - num in hashtable:
                return [hashtable[target - num],i]
            hashtable[num] = i

        return []
```

`hashtable = dict()`:哈希表创建语句
`for i,num in enumerate(nums)`:把 nums 的下标、值分别拿出来，遍历
`if target - num in hashtable`:这里是用 key 来判断
`hashtable[num] = i`:这里 num 作为 key，i 作为value





#### 方法三：排序加双指针

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        list = []
        
        for i in range(len(nums)):
            list.append((nums[i],i))
        # 按数字大小排序
        list.sort()
        # 左指针指向最小的数，右指针指向最大的数
        left = 0
        right = len(list) - 1
        # 只要左右指针没有相遇，就继续查找
        while left < right:
            # 取出左右两个数字
            sum = list[left][0] + list[right][0]
            # 如果两数之和等于 target，返回它们原来的下标
            if sum == target:
                return [list[left][1],list[right][1]]
            # 如果和小于 target，说明当前和太小，需要让左边数字变大
            if sum > target:
                right -= 1
            # 如果和大于 target，说明当前和太大，需要让右边数字变小
            if sum < target:
                left += 1
```

`append(参数)` 只会**把括号里的整体当成 1 个元素**塞进列表，list.append((nums[i],i))，插入的整体就是（nums[i],i）
`list[left][0]`:拿到 list 列表的第 left 个值，值是元组，在拿到元组里第 0 个数
